# La Cronología: Fallout

### 🔗 [Acceder a la página](https://instituto-politecnico-modelo.github.io/Miceli-Tabuada-EjerciciosFront/paginaMiceli/index.html)

</div>

Una página web interactiva que explora la historia y cronología de la saga de videojuegos **Fallout**, presentada con la estética característica de los **Pip-Boy** del universo falloutiano.

## 📋 Descripción

Este proyecto es un trabajo práctico individual para la materia **T. DE INNOVACIÓN TECNOLÓGICA**, desarrollado como parte del plan de estudios del Instituto Politécnico Modelo.

La página presenta información sobre tres títulos principales de la saga Fallout, destacando su contexto histórico, cronología y características clave dentro del universo postapocalíptico.

## 🎮 Características

- **3 páginas principales**: Home, Fallout, Fallout: New Vegas y Fallout 4
- **Diseño Pip-Boy**: Estética verde neón inspirada en los dispositivos del juego
- **Responsive**: Adaptado para dispositivos móviles y de escritorio
- **Navegación intuitiva**: Header sticky con enlaces a todas las secciones
- **Tipografía única**: Fuente Overseer personalizada para títulos
- **Variables CSS**: Paleta de colores y fuentes centralizadas

## 🛠 Tecnologías

- HTML5 con estructura semántica
- CSS3 (Mobile First)
- Fuentes personalizadas (@font-face)
- Material Symbols para iconos
